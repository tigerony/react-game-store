<h1 align="center">Shopping Cart</h1>

![](/assets/presentation.png)

## Built with
* React
* TypeScript
* Framer Motion
* React Router
* Sass

## Features
* Extensive catalog of games thanks to the [RAWG API](https://rawg.io/apidocs).
* Search any game you want.
* Games information and screenshots.
* Responsive design.
* Lazy loading images.
* _Smooth_ animations.
